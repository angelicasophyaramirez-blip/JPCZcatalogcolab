# Verification Outputs

This directory is for small human-readable summaries that confirm whether the detector and catalog are behaving reasonably.

Planned artifacts:

- `december_benchmark_summary.md`
  - total detected December events before subtype labels
  - comparison against Shinoda's approximately 35-event benchmark over 2000-2018
  - notes on any known mismatch
- `feb2018_baseline_summary.md`
  - whether an event was detected during `2-7 February 2018`
  - detected peak time
  - comparison against the expected `3 February 2018` peak

These are verification artifacts, not primary science outputs.
